Objective

As an entry-level web developer, your task is to translate a provided mockup from the design team into semantic HTML code. This assignment will test your knowledge and understanding with inserting a logo, creating menu navigation, and incorporating other content as outlined in the design. The primary focus is to correctly utilize HTML elements such as div, span, etc.

Instructions

  1. Develop a simple one-page website that mirrors the mockup design.
  2. Design a logo using appropriate web fonts, incorporating two specified colors from the design.
  3. Create a right-aligned menu navigation with links set to '#' and text color in either black or dark gray (e.g., <a href="#" style="color: #333;">).
  4. Utilize h1 to h6 tags and paragraph tags with suitable formatting to enhance the visual appeal of the content.
  5. Incorporate 3 images with varying dimensions ensuring consistent width and height. Alignment may vary, focusing on maintaining visual appeal.
  6. Apply a background pattern using an image from Subtle Pattern Links to an external site. and repeating it across the page as a background.
  7. Use an external CSS file named styles.css to organize and manage your styles.
  8. Verify and eliminate any errors in your code.
  9. Establish a repository for the project on GitHub. Regularly commit and push changes, avoiding the submission of zip files.

Project Submission

  1. Add your Git log in the comment area, highlighting key milestones and changes.
  2. Paste your GitHub repository URL as a comment.
